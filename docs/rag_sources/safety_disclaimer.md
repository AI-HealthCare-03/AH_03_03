# 안전 문구, 평가 제외, 진단 불가 후보 지식

- status: candidate_unreviewed
- source_org: 질병관리청 국가건강정보포털, 보건복지부
- source_url: https://health.kdca.go.kr
- runtime_use: safety_prompt_candidate

## 사용 목적

- 분석 결과, 식단 점수, 챗봇 답변, RAG 참고 정보에 공통으로 붙는 안전 문구 후보로 사용한다.
- 법률 자문 문서가 아니며, 실제 서비스 약관/의료 자문 문구는 별도 검토가 필요하다.

## 요약

- 이 서비스의 분석과 설명은 의료 진단, 확진, 치료, 처방, 약물 복용 변경 판단을 대신하지 않는다.
- 건강검진 수치, 식단 점수, ML 위험도는 사용자가 입력하거나 업로드한 정보에 기반한 참고용 건강관리 신호다.
- 검진값 하나만으로 질환을 특정하거나 정상/비정상을 단정하지 않는다.
- 검사값이 없거나 성별·연령별 검사 대상이 아닌 경우에는 평가 제외 또는 참고 불가로 안내한다.
- 증상, 반복적으로 높은 수치, 복약 변경, 치료 판단이 필요한 경우에는 의료진 상담을 권고한다.
- LLM 또는 RAG 답변은 입력에 없는 질환, 수치, 약물, 치료법을 임의로 추가하지 않아야 한다.
- 응급 증상, 심한 통증, 급격한 악화, 자해 위험이 언급되면 즉시 전문기관 또는 응급 도움을 안내한다.

## 공통 문구 후보

이 정보는 진단이 아니며 건강관리 참고용입니다. 정확한 진단과 치료, 약물 복용 변경은 의료진과 상담해 주세요.
