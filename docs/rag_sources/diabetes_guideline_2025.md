## 요약

- 업로드된 2025 당뇨병 진료지침 PDF는 RAG raw source 후보이며, repo에는 원문 PDF를 보관하지 않는다.
- MVP에서는 당뇨, 공복혈당, 당뇨병전단계, 의학영양요법, 혈압·지질·비만 동반관리 설명 후보로만 사용한다.
- 혈당 수치는 사용자 입력 또는 OCR 결과 기반의 참고 신호이며, 단일 수치만으로 진단 또는 확진을 단정하지 않는다.

- 혈압, 지질, 비만 동반관리는 만성질환 위험요인 설명에 참고할 수 있지만 치료 목표나 약물 처방처럼 표현하지 않는다.
- 검진값이 반복적으로 높거나 증상이 있으면 의료진 상담과 추가 검사를 권고한다.
- 이 요약은 원문 대조와 의료진 검토 전까지 `candidate_unreviewed` 상태로 유지한다.
