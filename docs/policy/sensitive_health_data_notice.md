# 민감 건강정보 안내문 초안

이 문서는 MVP 시연용 초안이며 법률 자문 문서가 아니다. 실제 서비스 출시 전에는 법률 검토와 개인정보 처리방침 본문 반영이 필요하다.

## 1. 민감 건강정보란

서비스에서 다루는 건강검진 수치, 혈압, 혈당, 지질 수치, 체중/BMI, 혈색소, 복약/영양제, 식단 기록, 챌린지 수행 기록은 개인의 건강상태를 추정할 수 있는 민감한 정보에 해당할 수 있다.

## 2. 수집 목적

민감 건강정보는 아래 목적을 위해 수집한다.

- 간편 건강 분석 제공
- 건강검진 기반 정밀 분석 제공
- 식단 질병군별 점수 계산
- 맞춤 챌린지 추천
- 복약/영양제 관리와 알림
- 사용자 본인의 건강관리 기록 확인

## 3. 분석 결과의 성격

서비스 분석 결과는 의료 진단, 확진, 치료, 처방이 아니다. CatBoost 모델, rule-based 계산, OCR, 식단 점수, 챗봇 응답은 건강관리 참고용 신호와 안내를 제공하기 위한 것이다.

증상이 있거나 수치가 우려되는 경우에는 의료기관 상담이 필요하다. 약물 복용, 중단, 변경 판단은 반드시 의료 전문가와 상의해야 한다.

## 4. 보관과 삭제

민감 건강정보는 서비스 기능 제공에 필요한 기간 동안 보관한다. 사용자는 삭제를 요청할 수 있으며, 법령상 보존 의무가 없는 정보는 삭제 또는 비식별화 처리 대상이 된다.

운영 진입 전에는 자동 삭제 주기, 보관 기간, 백업 보관 기간, 로그 retention 정책을 확정해야 한다.

## 5. 접근 제한

민감 건강정보는 사용자 본인과 필요한 권한을 가진 관리자만 제한적으로 접근할 수 있어야 한다. 관리자 화면에서는 원문 과다 노출을 피하고, 접근 로그를 남기는 정책을 적용한다.

## 6. 외부 provider 사용

OCR, LLM, SMS, 이메일 같은 외부 provider를 사용하는 경우 전송되는 정보 범위와 목적을 최소화해야 한다. 실제 운영 전에는 외부 위탁/제3자 제공 여부, 보관 여부, 파기 정책을 고지해야 한다.
