## 8. 주요 인사이트

- **데이터 확장 효과 극적** — v0.1(F1 0.40) → v2.1(F1 0.55) → v4(F1 0.60) 순차적 개선. FE보다 데이터 확장이 훨씬 효과적
- **당뇨 가족력 순서: 형제 > 모 > 부** — 생활환경 공유 효과. 고혈압(모>형제>부)과 다른 패턴
- **근력운동일수 Top 10** — 세 질환 중 당뇨에서만. 인슐린 저항성 개선 경로 포착
- **성별 SHAP 4위** — 데이터 확장 후 성별 기여도 대폭 상승
- **v4 FE 효과 없음** — 충분한 데이터에서 CatBoost가 이미 최적 패턴 학습
